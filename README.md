# Astro (formerly Atomic Boy) – Atoms & Elements Educational Game

**Help the astronaut build atoms by collecting protons, neutrons, and electrons across planets!**

Originally titled "Atomic Boy" in early videos, updated to "Astro" on the Legends of Learning platform.

### Learning Objective
- Understand **atomic structure**: Protons, neutrons, electrons, and how they form elements.
- Explore basic chemistry concepts: Building artificial atoms using an atom generator in space.

### Gameplay
- Travel to different planets to gather subatomic particles (protons, neutrons, electrons).
- Return to the atom generator to assemble stable atoms.
- Fun space adventure with educational feedback on element formation.
